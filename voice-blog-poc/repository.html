About This Repository

This project demonstrates a voice-first workflow that transforms spoken ideas into structured content and live page updates.

What It Includes

Audio recording from the browser, Whisper-based transcription, transcript-to-markdown formatting, and AI-guided site transformation using a JSON layout model.

Authentication and Sessions

Users can sign in with Google OAuth or local email and password accounts. Sessions protect application routes and API endpoints.

Publishing Workflow

The backend supports prompt-driven markdown generation and optional publishing workflows that can push content to GitHub.
